# 🚀 Chrome Extensions Pack

A collection of simple and useful Chrome extensions designed to improve user experience, accessibility, and productivity while browsing.

---

## 📦 Extensions Included

### 🎨 Background Color Changer
- Change the background color of any webpage
- Easy-to-use interface
- Enhances readability and personalization

---

### 🔠 Text Size Adjuster
- Increase or decrease text size on webpages
- Improves accessibility for users
- Quick and responsive controls

---

### 📝 Word Counter
- Counts words on a webpage or selected text
- Useful for writers, students, and developers
- Lightweight and fast

---

## 🛠️ Tech Stack

- JavaScript  
- HTML  
- CSS  
- Chrome Extension APIs
